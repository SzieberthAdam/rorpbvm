About this Module 

The Republic of Rome is my favourite boardgame of all time. Too bad that the complexivity and lenght of the game makes gamers to avoid from playing it. This is the reason I wanted to make a module which provides an easy overview over the game and reduces game length with useful features. Thus I came to the idea of implementing the nice structure which was made by RoR fans for PBEM play. This resulted the name PBVM (Play by VASSAL Module) which refers to the PBEM sources but pointing ou that this was made for real time play. VASSAL has a nicely working Save/Reload function which allows players to split the gametime (I recommend Revolution Phase for this).

I wanted to include most of the optional rules in the module. Some of them: Extra scenarios (Civil War, Latin/Italian periods), ageing, traitors, optional attacking order... More will be supported by next releases.

I wish you good luck and enjoyment by using this module!

Adam Szieberth
