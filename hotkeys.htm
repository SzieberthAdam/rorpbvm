Hotkeys

Hotkey Function
Ctrl+T Transform cards to markers and vica versa
Delete Delete a marker, or send card or marker to Discards, back to Mortality Chits, etc.
Insert Clone a marker
Shift+Arrow Move marker on main board
Up, Down, Num+, Num- Increment something (check with right click)
F1, F2, F3 Roll 1d6, 2d6, 3d6
F4 Show/Hide Markers
Ctrl+1 ... Ctrl+6 Show Faction hands

For other hotkeys check individual game piece.

Note that not all hotkeys work for all game pieces. Senators can not be cloned for example.
